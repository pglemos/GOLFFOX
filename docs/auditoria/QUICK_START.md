# Quick Start - Aplicar Correções

**⏱️ Tempo:** 2-5 minutos

---

## PASSO 1: Aplicar Script Consolidado

```
1. Supabase Dashboard → SQL Editor
2. Copiar: apps/web/database/migrations/000_APPLY_ALL_MIGRATIONS.sql
3. Colar e executar
```

## PASSO 2: Validar

```
1. Copiar: apps/web/database/scripts/validate_migrations.sql
2. Colar e executar
3. Verificar ✅ em todas as validações
```

## PASSO 3: Realtime

```
1. Dashboard → Database → Replication
2. Habilitar driver_positions
```

---

Todas as correções aplicadas.
